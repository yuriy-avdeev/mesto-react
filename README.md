Моя первая интерактивная веб-страница
Вы можете ознакомиться c подготовленной мною интерактивной веб-страницей по этой ссылке.
Данный проект выполнен в рамках учебного процесса совместно с Яндекс.Практикум с использованием возможностей HTML, CSS и JavaScript.

Приведенный проект решает две основные задачи:

Для меня это - закрепление на практике теоретических знаний;
Для пользователей - дает представление о уровне моих возможностей на данный момент.
Для работы с данной станицей вам не нужно иметь дополнительных библиотек либо программ.
